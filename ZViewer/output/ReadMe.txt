-- ZViewer 의 유용한 기능 --
 * 이미지를 캐시해두어 빨리 볼 수 있게 함. 50메가 캐시 메모리 제한.
 * 탐색기에서 오른쪽 클릭 -> ZViewer 로 클립보드의 이미지를 바로 파일로 붙여넣기 기능 추가 - 24Bit 형식으로만 저장
 * jpeg2000 이미지 지원
 * 이미지 빠르게 움직이기 - "ASZX", "7913" 키를 이용하여 현재 이미지의 '왼쪽위', '오른쪽위', '왼쪽아래', '오른쪽아래' 로 바로 갈 수 있음
 * 일본 서적을 스캔한 경우. 이미지를 넘길 때 오른쪽 위부터 보여주는 기능(팝업 메뉴로 설정가능)
 * 유니코드 지원으로 어느 언어의 윈도우에서나 사용 가능. 일본어로 된 폴더나 파일을 볼 수 있음

-- 기본 기능 --
 * 최적화된 속도로 빠르게 실행
 * 탐색기에서 오른쪽 버튼을 이용한 미리보기 기능
 * 다양한 이미지 파일 지원 : bmp, jpg, jpeg2000, png, tif, psd, gif, dds, tga, pcx, xpm, xbm, cut, ico, hdr, jng, koa, mng, pcd, ras, wbmp
 * 다른 형식의 그림 파일로 저장 기능
 * 파일 이동, 삭제 기능
 * 실수로 Home, Last 등을 눌러 처음, 마지막 파일로 갔을 때 Backspace 키를 이용한 이전 위치로 돌아가기
 * Ctrl+PageDown(다음 폴더), Ctrl+PageUp(이전 폴더) 를 통한 폴더간의 이동
 * 이미지 회전 기능. 시계방향(키 - ] ), 반시계방향(키 - [ )
 * 이미지 10장씩 넘기기. Ctrl+오른쪽화살표(다음 10장), Ctrl+왼쪽화살표(이전 10장)를 이용해서 여러장을 넘길 수 있음.
 * 언어팩을 이용한 국제화 - 현재 한글, 영어 자동 인식.
 * 바탕화면 배경으로 설정 기능
